DrugSaleAdministration README
=============================
##简介
暨南大学计算机系2009级数据库课程设计——药品销售管理系统
##工具

* Database: Microsoft Office Access
* Language: Java 1.6
* GUI: Java AWT
* Report Generator: JasperReport
